# SAMS

流浪动物管理系统服务端（其他有[页面](https://github.com/dmu-wxy/SAMS_WEB)、[小程序](https://github.com/dmu-wxy/samsMini)）

## 功能

- 管理员

    操作员管理（员工搜索删除、状态修改、权限管理），权限组管理（角色对应权限管理）

- 操作员

    流浪动物信息管理（增删改查、数据文件导入导出、高级搜索）

- 资金（待添加）

    资金记录

- 聊天（待添加）

    操作员、管理员之间聊天功能

## 技术栈

- springboot2.4.0
- springsecurity
- MySQL8
- Redis
## 搭建

创建一个sams数据库，执行doc文件夹下的sql脚本。
